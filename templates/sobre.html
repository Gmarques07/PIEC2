Água em Ação

Gerenciamento eficiente do abastecimento de água para cisternas em áreas rurais.

Sobre o Projeto

O projeto "Água em Ação" visa facilitar o gerenciamento do abastecimento de água para cisternas em áreas rurais, conectando comunidades necessitadas com empresas fornecedoras de água. Nosso sistema web permite o registro de pedidos, acompanhamento de entregas e gerenciamento eficiente de informações.

O Problema

  • Falta de gerenciamento eficiente de cisternas e poços em áreas rurais
  • Alto índice de perdas na distribuição de água no Nordeste (50,9% em Pernambuco)
  • Escassez de cisternas prejudica o armazenamento de água em períodos chuvosos
  • Em média, 200 famílias no município necessitam de abastecimento regular

Nossa Solução

Um sistema web com as seguintes funcionalidades:

  • Cadastro de empresas e usuários
  • Criação e edição de pedidos de água
  • Comunicação entre empresa e usuário
  • Administração de pedidos pela empresa
  • Integração com sensoriamento e visão computacional

Tecnologias Utilizadas

Backend

Flask (Python) para API e gerenciamento de dados

Frontend

HTML, CSS, Bootstrap

Banco de Dados

SQL

Visão Computacional

OpenCV para detecção de objetos e rachaduras

IoT

Sensores de nível de água, pH e cloro utilizando Arduino
